How to Use

  1. Run the tool and select your combo file (.txt format)
  2. Choose the number of threads (1-200)
  3. Monitor real-time progress
  4. Results will be saved in the "Result" folder

Performance Tips

  • Optimal Thread Count: Start with 50 threads and adjust based on your system's performance
  • File Size: For best results, process files under 1GB at a time
  • Regular Updates: Always use the latest version for best performance and features
